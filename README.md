# Dong2026-GRB-X-Ray-Plateau-Data

This data set contains 227 gamma-ray bursts with X-ray plateau emission observed by Swift/XRT from 2005 March to 2025 August. 
The sample corresponds to the GRB X-ray plateau data used in Dong et al. (2026), https://iopscience.iop.org/article/10.3847/1538-4357/ae66f1.

The data set includes both long and short GRBs. The 11 short GRBs in the sample have been explicitly flagged in the table.

Users of this data set should cite the following references:

Tang, C.-H. et al. 2019, ApJS, 245, 1

Deng, C. et al. 2023, ApJ, 943, 126

Dong, X.-F. et al. 2026, ApJ, 1003, 227
